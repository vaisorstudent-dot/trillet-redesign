## Unit Economics

**Baseline COGS:** $0.046/min (without premium voices)

| Tier | Per-Minute Charge | COGS/Min | Margin/Min | Per-Min Gross Margin |
|------|-------------------|----------|------------|---------------------|
| Starter (included) | $0.59 effective ($59/100min) | $0.046 | $0.544 | **92%** |
| Starter (overage) | $0.20 | $0.046 | $0.154 | **77%** |
| Agency / Lite SaaS | $0.12 | $0.046 | $0.074 | **62%** |
| Pro SaaS | Per-endpoint (varies) | $0.046 | Varies | 60-70%+ |

**Competitive position:** $0.12/min is still significantly cheaper than Vapi, Retell, and Bland — even after the upcoming price increase from $0.09.

## Revenue Model: Dual-Stream

**1. Subscription revenue** — base monthly fee
- For Agency/Lite SaaS/Pro SaaS: nearly 100% margin (no minutes included)
- Predictable, recurring

**2. Usage revenue** — per-minute charges
- 62-77% gross margin
- Scales automatically as agencies deploy more agents handling more calls
- Compounds with growth (more agencies × more endpoints × more minutes)

**Key insight:** Every new endpoint an agency adds generates incremental high-margin usage revenue for Trillet without additional sales effort.

## Tier Differentiation = Deployment Model

The same AI runs at every tier. Tiers are differentiated by deployment wrapper, not feature gates:

| Tier | Deployment Model |
|------|-----------------|
| Starter | 1 operator → self-setup → 1 agent → 1 business |
| Agency | 1 operator → manual config per client → N agents |
| Lite SaaS | 1 operator → template library → one-click deploy → clients self-onboard |
| Pro SaaS | 1 operator → custom template → API deployment → entire network auto-configures |
